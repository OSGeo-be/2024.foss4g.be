Setting up a boundaries database with PostGIS topology

Benjamin Deswysen
Engels
Woensdag 25/9, 13h00 - 17h00
EHBO

Hand's on workshop in the postgis topology ecosystem based on a previous session organized with OsGeo BE in 2023 (https://bdeswysen.gitlab.io/postgis_topology_workshop)

How to: setup a postgis topology database, ingest geographic data, convert geometries to topogeometries, simplify geometries using a topological model.

Open discussions about: advantages/limitations of using a topologic model in database, of postgis topology itself, ...

Requirement: a personal laptop with a (working) postgis database and, if possible, "postgis topology" extension already installed or postgres user access OR a docker environment
